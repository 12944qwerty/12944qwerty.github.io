Bookmarklets

Just some quick and easy things you can use to make WoX efficient. Drag each link into the bookmark bar.
  • Cuddle All Pets - Go to someone's pets and click this bookmarklet. It will cuddle all of them!
  • Welcoming Bookmarklet! - Just fill in the text below, and click the bookmarklet when on the user page. Vala!


    Bookmarklet here: Welcome Users
  • Speak Backwards - Type what you want to into the prompt and it should type it backwards where you want it to.
  • Deposit All - This one deposits all money you have in your pocket into the bank, assuming you're on the bank's page.

TamperMonkey

Or, instead of using bookmarklets.
  1. You can install TamperMonkey.
  2. Click the extension on the top-right, and click Dashboard.
  3. Next, go and copy the contents.
  4. Create a userscript at the dashboard and replace the contents with the contents from the the link you want to use.
  5. Repeat for every script.

  • Cuddle All Script
